1. What is HTML and why is it called a markup language?

Ans.   HTML stands for HyperText Markup Language. It is used to create webpages and structure content like headings, paragraphs, images and links.

It is called MarkUp because it uses tags to descibe the content on the webpage

2. What is the difference between HTML and HTML5?

Ans.   HTML is the older version used to create webpages

HTML5 is the latest version with new features like:

  • audio and Video support
  • Semantics tags
  • Better multimedia support
  • improved forms and API's

HTML5 makes webpages more modern and Interactive.

7. What are meta tags and why are they used?

Meta tags provide information about the webpage to browsers and search engines. They are placed inside the section of an HTML document.

Meta tags are used to define the character encoding, make webpages responsive on different devices, improve SEO, provide webpage descriptions and store author information.

8. Why do we use the <source> tag inside audio and video tags?

The <source> tag is used inside audio and video elements to provide media files in different formats. Different browsers support different file types, so using multiple source tags helps the media work properly in more browsers.

9. What is iframe and where is it used in real-time applications?

An iframe is used to embed another webpage or external content inside a webpage. It is commonly used in real-time applications to embed YouTube videos, Google Maps, online forms, advertisements and social media content.

10. What is the difference between block-level elements and inline elements?

Block-level elements always start on a new line and take up the full width available. Examples include <div> , <p> and <h1>. Inline elements do not start on a new line and only take up the width needed for their content. Examples include <span>, <a> and <strong>.
